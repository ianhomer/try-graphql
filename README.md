# Try GraphQL

## Getting started

    pnpm install
    pnpm start

and access <http://localhost:8080/graphql>

## End to end test

    pnpm start
    pnpm test:e2e

To see debug logs

    export DEBUG=pw:webserver

To enable CI mode

    export CI=true
